What is Cascade Collab?

Cascade Collab is a collaborative art project where multiple artists contribute to a vertically stacked artwork, each drawing a segment in their own unique style. Inspired by the iconic Durarara! ending meme, the goal is to showcase artistic diversity in a seamless flow.

Want to Learn More?

Whether you're a well-known artist with a big following or a niche artist with just a few happy followers, this project welcomes you!
